# Restaurant-Failure-Prediction-Using-Yelp-Review-Data
Machine learning models predict restaurant failure with 88-95% precision across Philadelphia, Tampa, and Santa Barbara. Key finding: time since last review (0.73 correlation) dominates predictions while star ratings are irrelevant (-0.05). "Restaurants fail in silence, not in bad reviews." Analyzes 1.15M reviews for 9,579 restaurants.
